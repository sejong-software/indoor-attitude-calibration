## 사용 언어
- C#, matlab, unity

## 목표
- 영상 정보를 활용한 정확한 방향각 계산
- 방향각의 오차를 추정 및 보상하는 알고리즘 개발
- 시뮬레이터를 통한 알고리즘 성능 검증

## 주요 연구 내용
- IMU 및 지자기 센서를 이용한 자세 계산 알고리즘 개발
- 영상 정보를 활용한 방향각 오차 보상 알고리즘 개발
- 자세 계산 및 방향각 오차 보상 알고리즘 성능 검증을 위한 시뮬레이터 개발
